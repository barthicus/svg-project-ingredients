This demo is an example of using library Svg Project Ingredients. It is a JS library that allows visualise product ingredients as SVG object.
More information about library, methods, emitted events can be found on project repo.

Init library:

import SvgProductIngredients from './SvgProductIngredients'
const svgElement = document.querySelector('.svgElementClass')
const productViewer = new SvgProductIngredients(svgElement, 'assets/apple.png', {width: 850, height: 500})

Select slide on ingridient (circle) click using emitted event:

productViewer.on('ingredients.select', ingredientNr => selectSlide())

Select ingridient from JS usign library method:

productViewer.selectIngredient(nr)
